# Schedule Auditory

Небольшое Flask-приложение для просмотра свободных аудиторий на текущую пару.

## Что сделано

- Добавлена веб-страница с автопроверкой при открытии.
- Текущая пара определяется автоматически по фиксированной сетке:
  `08:45-10:05`, `10:20-11:40`, `11:55-13:15`, `13:30-14:50`,
  `15:05-16:25`, `16:40-18:00`, `18:15-19:35`, `19:50-21:10`.
- Добавлен API `/api/free-classrooms`.
- Если список аудиторий пустой, приложение показывает все аудитории, найденные в расписании.

## Что удалось выяснить

- Прямой поиск аудиторий через `schedule.siriusuniversity.ru` и Livewire оказался ненадежным:
  запросы вида `set("305")` возвращали пустое расписание без ошибки.
- На сайте есть отдельная страница `/classroom`, но для основной задачи удобнее использовать готовый JSON API из Android-приложения.
- В Android-коде найден рабочий источник данных:
  `https://eralas.ru/api/groups`
  `https://eralas.ru/api/schedule?group=...&week=0`
- В ответе расписания есть поле `classroom`, поэтому свободные аудитории считаются по расписаниям всех групп.

## Как теперь работает логика

1. Загружается список групп из `eralas.ru/api/groups`.
2. Для групп загружается расписание через `eralas.ru/api/schedule`.
3. Из всех занятий собираются аудитории.
4. Для текущего временного слота определяется, какие аудитории заняты, а какие свободны.
